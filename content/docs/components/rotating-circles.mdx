## Props

| Prop            | Type             | Description                                                                      | Default                                            |
| --------------- | ---------------- | -------------------------------------------------------------------------------- | -------------------------------------------------- |
| icons           | string[]         | An array of icons URLs to display in the rotating circles.                       | `defaultIcons` (an array of predefined image URLs) |
| children        | React.ReactNode` | The content to be displayed inside the box.                                      |                                                    |
| className       | string           | Additional CSS classes to be applied to the box content.                         |                                                    |
| durationInner   | number           | The duration (in seconds) for the inner circle's rotation animation.             | `1`                                                |
| durationMiddle  | number           | The duration (in seconds) for the middle circle's rotation animation.            | `1.2`                                              |
| durationOuter   | number           | The duration (in seconds) for the outer circle's rotation animation.             | `1.3`                                              |
| easeInner       | string           | The easing function for the inner circle's rotation animation.                   | `"linear"`                                         |
| easeMiddle      | string           | The easing function for the middle circle's rotation animation.                  | `"easeInOut"`                                      |
| easeOuter       | string           | The easing function for the outer circle's rotation animation.                   | `"easeInOut"`                                      |
| translateInner  | number           | The translation distance (in pixels) for the inner circle's rotation animation.  | `160`                                              |
| translateMiddle | number           | The translation distance (in pixels) for the middle circle's rotation animation. | `230`                                              |
| translateOuter  | number           | The translation distance (in pixels) for the outer circle's rotation animation.  | `320`                                              |
